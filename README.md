##Features 
-**Privacy Policy:** Tells how we will protect our users data and privacy.
-**Our Services:** Tells what kind of services we are providing to our users.
-**About Us:** Tells what kind of work we do in our organization and what we are.
-**Login:** After autehntication, lets the user login if an authorized user.
-**Signup:** If a user is not regristered one, let him allow to signup after collecting necessary information.
-**Reset Password:** If a user forgot his password let him reset it through his email.

##How it works
-If you are a new user:
 -You need to register first by providing your Name,Company mail, and Password.
-If you are an old user;
 -Just write down your email and password and after validating you are set to go.

##Security and validation
-**Strong Password:**Ask you to enter at least 8 characters, one uppercase,one special character, and at least one number.
-**Company based:** User must sign only using a valid company mail othewise not allowed(Gmail,Yahoo etc not allowed)
-**Password Input Masking:** When you input password, it shows in asterik"*" form

##Error handling:
-Shows an error message if wrong credentials.
